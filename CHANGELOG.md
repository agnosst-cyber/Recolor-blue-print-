# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.1.0] - 2025-01-13

### Changed
- Улучшена логика перекрашивания: объекты теперь перекрашиваются в разные оттенки голубого цвета #2A7DC7
- Сохранена различимость объектов: более светлые исходные объекты получают более светлые оттенки, темные - более темные
- Автоматическое распределение оттенков от светлого к темному на основе исходной яркости объектов
- Палитра оттенков варьируется от 15% до 85% яркости для максимальной различимости

### Technical
- Добавлены функции конвертации RGB ↔ HSL для работы с оттенками
- Реализован алгоритм сбора и сортировки объектов по яркости
- Улучшена обработка объектов с заливками и обводками

## [1.0.0] - 2025-01-13

### Added
- Базовая функциональность перекрашивания объектов в голубой цвет #2A7DC7
- Рекурсивная обработка всех дочерних элементов во фреймах и группах
- Перекрашивание заливок (fills) и обводок (strokes)
- UI интерфейс с кнопкой "Recolor"
- Автоматическая подстройка высоты окна плагина под контент
- Поддержка выделения фреймов, объектов и групп
- Уведомления о результате операции

### Changed
- Размер UI плагина: ширина 320px, высота автоматически подстраивается
- Стили интерфейса приведены в соответствие с дизайн-системой Figma:
  - Использование шрифта Inter
  - Цветовая палитра Figma
  - Отступы по 8px сетке
  - Правильные размеры шрифтов (11px для текста, 13px для заголовков)
  - Focus состояния для доступности

### Technical
- TypeScript конфигурация
- Сборка проекта через npm scripts
- Настроен .gitignore для исключения node_modules и логов
- README.md с инструкциями по установке и использованию
- Проект загружен в GitHub репозиторий
