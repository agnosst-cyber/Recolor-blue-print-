# Recolor Plugin для Figma

Плагин для автоматического перекрашивания объектов в оттенки голубого цвета #2A7DC7.

## Описание

Плагин позволяет быстро перекрасить все объекты во фрейме, группе или выделенном объекте в голубой цвет #2A7DC7. Перекрашиваются как заливки (fills), так и обводки (strokes) всех дочерних элементов.

## Как использовать

1. Запустите плагин в Figma: `Plugins` → `Development` → `Recolor`
2. Выделите фрейм, объект или группу
3. Нажмите кнопку "Recolor"
4. Все объекты будут перекрашены в голубой цвет #2A7DC7

## Установка для разработки

### Требования

- Node.js (версия 14 или выше) - [скачать](https://nodejs.org/)
- npm (устанавливается вместе с Node.js)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd Recolor
```

2. Установите зависимости:
```bash
npm install
```

3. Скомпилируйте TypeScript:
```bash
npm run build
```

4. Откройте Figma Desktop App

5. Импортируйте плагин:
   - Перейдите в `Plugins` → `Development` → `Import plugin from manifest...`
   - Выберите файл `manifest.json` из папки проекта

## Разработка

### Команды

- `npm run build` - компиляция TypeScript в JavaScript
- `npm run watch` - автоматическая компиляция при изменении файлов
- `npm run lint` - проверка кода линтером
- `npm run lint:fix` - автоматическое исправление ошибок линтера

### Структура проекта

- `code.ts` - основной код плагина (TypeScript)
- `code.js` - скомпилированный JavaScript (генерируется автоматически)
- `ui.html` - интерфейс плагина
- `manifest.json` - конфигурация плагина
- `package.json` - зависимости проекта

## Технические детали

- **Размер UI**: 320px ширина, высота автоматически подстраивается
- **API версия**: 1.0.0
